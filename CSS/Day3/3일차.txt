요소 속성  속성값

[속성명]: 속성 이름으로만 선택
[속성명='속성값']: 속성 이름과 속성값으로 선택 
[속성명~='속성값']: 속성값이 완벽히 일치하는 문자를 가지는 요소를 선택
[속성명*='속성값']: 속성값이 포함된 모든 문자를 가지는 요소를 선택
[속성명^='속성값']: 속성값이 접두사로 시작되는 모든 문자를 가지는 요소를 선택
[속성명 |='속성값']: 속성값이 접두사로 시작하는 문자 중에서 완벽히 일치하는 문자 또는 '-'으로 연결된 문자를 가지는 요소를 선택
[속성값 $='속성값']: 속성값이 접미사로 끝나는 모든 문자를 가지는 요소를 선택 

가상 클래스와 가상 요소 

앵커 가상 클래스
앵커의 요소에 특정 디자인을 입혀주는 동적인 스타일을 만든다. 
:hover : 사용자가 해당 요소에 커서를 올렸을때 
:active : 요소가 활성화 되었거나 클릭했을때 
:focus : 해당 요소에 키보드 포커스가 맞춰졌을때
:visited: 사용자가 해당 링크를 다녀왔을때
:link    : 해당 요소에 하이퍼링크가 적용됐을때 

순서에 따른 가상 클래스
CSS 스타일을 주기 위해 클래스를 추가할 필요없이 요소 중에서 순서에 따라 원하는 특정 요소를 선택해 줄수있습니다.
예)
<ul>
  <li>김사과</li>
  <li>오렌지</li>
  <li>이메론</li>
  <li>바나나</li>
</ul>

li:first-child  -> li 요소중에서 첫번째 해당하는 요소에 CSS를 적용합니다. 
li:last-child  -> li요소중에서 마지막에 해당하는 요소에 CSS를 적용한다.
li:nth-child(n) -> li요소중에서 n 번째 요소에 CSS를 적용합니다.
li:nth-child(odd) -> li요소중에서 홀수번째 요소에 CSS를 적용한다. 
li:nth-child(even) -> li요소중에서 짝수번째 요소에 CSS를 적용한다.

가상 요소
가상 클래스가 실제로 존재하는 요소에 클래스 추가 없이 디자인을 입히기 위한 클래스라고 한다면, 가상 요소는 실제로 존재하지 않는 요소를 만들어 준다. 
예)
:first-line -> 해당 가상 요소는 텍스트의 첫번째 줄에만 특정 스타일을 입혀준다. 
:first-letter -> 텍스트의 첫문자에 특정 스타일을 입혀준다. 
:before -> 보통 이미지나 글, 그라데이션 등을 앞에 삽입시 사용한다.
:after  -> 보통 이미지나 글, 그라데이션 등을 뒤에 삽입시 사용합니다.

CSS의 컬러
1. 색상 이름으로 표현
red, blue, yellow, skyblue...

2. RGB 색상값으로 표현
모니터(스크린)은 빨간색, 녹색, 파란색을 혼합하여 색을 표현한다. HTML(CSS)에서도 이 세가지 색을 조합하여 색을 표현하는 RGB색상을 사용합니다. RGB색상은 기본색(RED, GREEN, BLUE)를 각각 0~255까지 범위를 가지고 표현합니다. 
예)
h2 {color:rgb(0,255,0);}  /*녹색*/

h2 {color: rgba(0,255,0,0.5);}  /*녹색 반투명*/
* alpha(투명도): 범위 ->0투명 ~ 1불투명

3. 16진수 색상값으로 표현
16진수 색상값은 RGB 색상값을 각각 16진수로 표현한 것입니다. 기본색(RED, GREEN,BLUE)을 각각 00부터 FF까지 범위를 가지고 표현한다. 예)
h2 {color: #00FF00;} /*녹색*/ -> h2 {color: #0F0; }

*<body>태그에 명시된 color 속성값은 웹페이지 내의 모든 텍스트 요소에 적용됩니다. 하지만 각 요소별로 따로 명시된 color 속성값이 있다면 해당 속성값이 <body>태그의 명시된 속성값보다 우선 적용된다.

CSS의 텍스트

color
텍스트의 색상을 설정합니다. 웹페이지 내에서 텍스트의 기본 색상은 검정색이다.

letter-spacing
텍스트 내에서 글자 사이의 간격을 설정한다. 

word-spacing
텍스트 내에서 단어 사이의 간격을 설정한다. 

text-align
텍스트의 수평 방향 정렬을 설정합니다. (left, right, center, justify)

text-indent
단락의 첫줄에 들여쓰기를 할지, 안할지를 설정한다. 웹 페이지에서 단락은 기본적으로 들여쓰기가 설정되어있지 않다. 

line-height
텍스트의 줄 간격을 설정한다.

text-devoration
텍스트의 여러가지 효과를 설정하거나 제거하는데 사용한다.  (none, underline, line-through, overline...)

text-transform
텍스트에 포함된 영문자에 대한 대소문자를 설정합니다. (uppercase, lowercase, capitalize)

font-variant
소문자를 작은 대문자 즉, 소문자 크기의 대문자로 설정한다. (small-caps)
